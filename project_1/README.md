# 🚀 Decode Labs Internship Projects

## 🧹 Project 1: Data Cleaning & Preparation

This project is part of the **Decode Labs Data Analytics Internship Program**.

---

## 🎯 Objective

The goal of this project is to clean and prepare raw data using **Python** and **Pandas** by:

✅ Handling missing values  
✅ Removing duplicate records  
✅ Standardizing date formats  
✅ Cleaning text columns  
✅ Improving data quality and integrity

---

## 🛠 Technologies Used

- 🐍 Python
- 📊 Pandas
- 📁 OpenPyXL
- 💻 VS Code

---

## 📂 Files Included

- 📄 `Decode_labs_dataset.xlsx` → Original raw dataset
- 🧠 `clean_data.py` → Python script for data cleaning
- ✅ `cleaned_dataset.xlsx` → Final cleaned dataset

---

## 🔄 Steps Performed

1. 📥 Loaded Excel dataset using Pandas
2. 🔍 Identified missing/null values
3. 🧩 Filled missing CouponCode values
4. 🧹 Removed duplicate records
5. 📅 Converted date columns into standard format
6. ✨ Standardized text formatting
7. 💾 Saved cleaned dataset into a new Excel file

The cleaned dataset was successfully generated with improved consistency and data integrity ✅
