# SQL Data Analysis on E-Commerce Dataset

## Objective

Analyze an e-commerce dataset using SQL queries to extract meaningful business insights.

## Tools Used

- SQL
- SQLite
- VS Code
- Python
- Excel Dataset

## SQL Concepts Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT
- SUM
- AVG

## Analysis Performed

1. Counted total orders.
2. Calculated total revenue.
3. Calculated average order value.
4. Filtered high-value orders.
5. Sorted orders by total price.
6. Analyzed payment methods.
7. Analyzed referral sources.
8. Calculated revenue by referral source.
9. Identified top-selling products.
10. Generated business insights from order data.

## Conclusion

This project demonstrates SQL data analysis skills by performing filtering, sorting, grouping, and aggregation operations on an e-commerce dataset.
