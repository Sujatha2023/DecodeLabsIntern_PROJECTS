# 📊 Exploratory Data Analysis (EDA) Project

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an e-commerce dataset using Python.

The main goal of this project is to analyze the dataset, identify patterns, trends, distributions, correlations, and outliers, and generate meaningful business insights.

---

# 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

---

# 📂 Dataset Information

The dataset contains e-commerce order details such as:

- Order ID
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Referral Source
- Order Status
- Total Price

Dataset Type:

- Excel Dataset (`.xlsx`)

---

# 📈 Exploratory Data Analysis Performed

## ✅ Data Cleaning

- Checked missing values
- Verified data types
- Explored dataset structure

## ✅ Descriptive Statistics

Calculated:

- Mean
- Median
- Maximum
- Minimum
- Count

## ✅ Data Visualization

Created visualizations for:

- Payment Method Distribution
- Referral Source Distribution
- Total Price Distribution
- Product Analysis
- Correlation Heatmap
- Outlier Detection

## ✅ Outlier Detection

Used boxplots to identify unusual TotalPrice values.

## ✅ Correlation Analysis

Analyzed relationships between numerical columns using a heatmap.

---

# 📊 Key Insights

- Most customers used a few popular payment methods frequently.
- Some TotalPrice values were extremely high, indicating outliers.
- Referral sources like Instagram and Email generated many orders.
- Quantity and TotalPrice showed positive correlation.
- Product sales distribution was uneven across products.
